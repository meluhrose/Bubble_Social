## Description

Bubble is a social media application where users can create posts, manage their profiles, and connect with other users. The application uses the Noroff Social API v2 for backend services.

## Features 

- **User Authentication**: Register and login with Noroff student email
- **Feed**: Browse posts with infinite scroll pagination
- **Search**: Real-time post search functionality
- **Create Posts**: Share content with title, body, and optional images
- **Edit/Delete Posts**: Full CRUD operations for your own posts
- **User Profiles**: View and edit your profile with bio and avatar
- **Follow System**: Follow/unfollow other users
- **Post Details**: View individual posts with full information
- **Responsive Design**: Works across different screen sizes

## Getting Started‚≠ê

### Prerequisites

- A modern web browser
- A Noroff student email (@stud.noroff.no)
- A local development server (e.g., Live Server extension for VS Code)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/meluhrose/Bubble_Social.git
cd Bubble_Social
```

2. Open the project in your code editor

3. Start a local development server

4. Open the application in your browser

### Usage

1. **Register**: Create an account using your @stud.noroff.no email
2. **Login**: Sign in with your credentials
3. **Navigate**: Use the navigation bar to access different sections
4. **Create Posts**: Share your thoughts with the community
5. **Interact**: Browse and search through other users' content

## API Documentation üìù

The application uses the [Noroff Social API v2](https://v2.api.noroff.dev/social).

### Key Endpoints:
- `/auth/register` - User registration
- `/auth/login` - User login
- `/auth/create-api-key` - API key generation
- `/social/posts` - Posts management
- `/social/profiles` - Profile management
